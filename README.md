## 去中心化代幣交易平臺

立即開始你的數位資產交易：無需註冊帳號，連接錢包即可交易。

真正的去中心化交易平臺，用戶無需註冊，集合了現時區塊鏈最流行的玩法，集兌換、流動性挖礦、跨鏈交易、IDO於一體，打造Nexus生態鏈最具活力的業務形態。

### 去中心化

去中心化直接使用您的錢包App開始交易。 不同於傳統的中心化交易所，NexusSwap在交易時不會持有您的資金：**您對自己的數位貨幣有100%的控制權。**

### 費用低廉

費用低廉何必多花冤枉錢？ 對比乙太坊或比特幣，NSC擁有低廉的交易成本。 NexusSwap的交易手續費也遠比其他頂級去中心化交易所低！

### 使用Syrup資金池賺取代幣

質押NU，免費賺取代幣。 是不是很簡單？？\
質押代幣免費賺取代幣。 糖漿池將持續為您賺取收益。 自動NU池子還會為您自動進行複投，產生複利，收益更為可觀。 每天都有新的項目不斷加入，讓您不僅賺得更多，而且賺得更久。

### 使用農場賺取NU

投入LP代幣，賺取NU。 與Syrup資金池相比，您或許要承擔更多的市場波動風險。 但更高的年利率，能為您减少市場波動帶來的影響。

### 賺取交易手續費

您感興趣的代幣對沒有農場？ 沒問題。 即使交易對沒有開設農場，當您把代幣押在流動性池（LPs）時，您仍然可以賺得交易手續費。
