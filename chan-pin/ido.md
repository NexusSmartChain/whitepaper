使用NU-NEX LP代幣（流動性憑證）購買新的代幣。

項目方得到NEX，NexusSwap銷毀NU。**你得到新的代幣。**

1. **您需要NU-NEX LP代幣（通過添加流動性得到），**來參加IDO；
2. **募集到資金的一半，也就是NU的那一部分，將被銷毀。** 例如：若最後募集得的資金總數為一百萬，則大約價值五十萬的NU將被銷毀。

### 如何参加IDO？

**開售前**

1. 購買NU和NEX代幣。
2. 添加NU-NEX流動性以獲得NU-NEX LP代幣（流動性憑證）。

**開售後：**

1. 銷售開始後，訪問 [https://nexusswap.io/ido](https://nexusswap.io/ido) 並投入您的NU-NEX LP代幣，來購買IDO項目代幣。

**完售後：**

1. 銷售結束後，您可以回到IDO頁面領取您購得的代幣。
2. 銷售完成後， 如果出現超售，您需要回到IDO頁面手動領取剩餘沒有用到的NU-NEX LP代幣。

{% hint style="info" %}
如果您决定遠在IDO開始前，即獲得NU-NEX LP代幣。 您可以將它們質押在NU-NEX農場，在等待IDO開售的同時，賺取NU獎勵。
{% endhint %}

{% hint style="warning" %}
當您持有NU-NEX LP代幣，您會有無常損失的風險。
{% endhint %}

## 什麼是「超售溢出」銷售方法？ <a href="#overflow" id="overflow"></a>

简单来说：您投入得更多，您将获得更多。在完售后，您可以领取任何没有成功使用并支付的 LP 代币。

在「超售溢出」方法中，用户可以根据自己实际情况投入任何数量的 LP 代币参与 IDO。最终获得的代币数量，将取决于他们投入资金量，占整个 IDO 投入资金量中的份额。用户在销售结束后，认领购得的代币时，将同时领回任何剩余的资金（LP 代币）。

{% hint style="warning" %}
若您投入的數量僅占總數非常小的一部分，您可能最終會買不到任何的IDO代幣。 因為您的投入數相對於總量來說非常小，以至於在計算時，您沒有資格獲得任何代幣。

在這種情況下，您仍可以在完售後，認領未消費的NU-NEX LP代幣。
{% endhint %}

## 如果IDO募集目標沒有達到，會怎麼樣？

銷售及後續工作將如期進行，所有募集得的資金都會被消耗。
